Scroll Animation

Step One:

Add this one line of js we can set a varible for what percentage of the page has been scrolled.

        
    window.addEventListener('scroll', () => {
    document.body.style.setProperty('--scroll', window.pageYOffset / (document.body.offsetHeight - window.innerHeight));
    }, false);

Step Two:

Add the following css to any animation to make them be paused and only move in correspondence to the page scrolling.

        
            animation: rotate 1s linear infinite;
            animation-play-state: paused;
            animation-delay: calc(var(--scroll) * -1s);
          
            animation-iteration-count: 1;
            animation-fill-mode: both;
